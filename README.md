
# Lumina Beauty Dashboard

A modern skincare intelligence web app that delivers personalized routines and product recommendations based on user preferences.


# Features

- Interactive skincare quiz (skin type, concerns, goals)
- Personalized skin analysis dashboard
- Smart product recommendation system
- Favorites system (saved in localStorage)
- Search & filter product explorer
- Dark / Light theme toggle
- Modal product details view
- Toast notifications for user feedback


# Tech Stack

- HTML5
- CSS3 (Custom properties, responsive design, animations)
- Vanilla JavaScript (ES6+)
- LocalStorage API

- Fully responsive UI
- Smooth animations & transitions
- Component-based JS structure (no frameworks)
